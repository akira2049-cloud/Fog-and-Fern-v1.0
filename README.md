# 🌿 Fog & Fern SF — Plant Shop & E-Commerce Management System

A modern, full-stack **Plant Shop & E-Commerce Management System** built with **Node.js**, **Express.js**, **EJS**, **SQLite**, and **Bootstrap 5**. The application provides a complete online shopping experience for customers while offering a powerful admin dashboard to manage products, inventory, orders, customers, and store operations.

Designed as a professional portfolio project, the system demonstrates modern web development practices, responsive UI design, secure authentication, session management, and dynamic database-driven content.

# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/fog-and-fern-sf.git
```

Go to the project folder:

```bash
cd fog-and-fern-sf
```

Install dependencies:

```bash
npm install
```

Create a `.env` file:

```env
PORT=3000
SESSION_SECRET=your_secret_key
NODE_ENV=development
```

Start the development server:

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```
